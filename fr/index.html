mdash

English

mdash est une alternative esthétique et pratique
à la page Nouvel onglet de Google Chrome



Installation

Installation

  • Cliquer sur le bouton Installer l'extension ci-dessus.
  • Autoriser Chrome à télécharger le fichier .crx en cliquant sur Continuer dans la barre de téléchargement.
  • Cliquer sur le bouton Installer dans la fenêtre qui s'ouvre ensuite.

Usage

Pour commencer

Une fois l'extension installée, ouvrez simplement une nouvelle fenêtre pour activer l'extension.
Un dossier nommé [Dashboard] sera alors créé dans le dossier Autres favoris.

Ajouter une section

Pour ajouter une section au dashboard, ouvrez Favoris → Gestionnaire de favoris puis naviguez jusqu'au dossier /Autres favoris/[Dashboard]/
C'est dans ce dossier que seront stockés les sections et favoris apparaissant dans le dashboard.

Chaque section est représentée par un dossier. Comme il n'est pas possible de joindre des méta-données à un dossier, c'est son nom qui sera utilisé pour déterminer dans quelle colonne il doit s'afficher.

Les dossiers préfixés par + seront affichés dans la colonne de gauche, ceux préfixés avec - le seront dans la colonne de droite.

Ajouter un favori tout en naviguant sur le Web

Allez sur le site que vous désirer ajouter au dashboard et cliquer sur l'étoile située à la droite de la barre d'adresse, puis choisissez un des dossiers précédemment créés.


Ajouter un favori depuis le dashboard

Cliquez sur le lien edit dans le coin en haut à droite (ou maintenez la touche Alt enfoncée), puis cliquez sur le gros bouton + qui apparaît dans chaque section.




Donnez un titre et une URL au favori (cette dernière doit commencer avec http:// ou un autre schéma), et cliquez sur Add (ou appuyez sur Enter).



Quand vous aurez fini d'ajouter des favoris, cliquez sur done en haut à droite de la page.

Editer ou supprimer un favori

Cliqez sur edit en haut à droite du dashboard (ou maintenez la touche Alt enfoncée), puis cliquez sur le favori que vous souhaiter modifier.
Dans la boîte de dialogue qui s'ouvre alors, faites les modifications nécessaires puis cliquez sur Save (ou appuyez sur Enter).
Pour supprimer un favori, cliquez sur Remove.



Quand vous aurez fini d'éditer vos favoris, cliquez sur done dans le coin en haut à droite.

Fonctionnalités prévues

  • Ajouter une section directement depuis le dashboard.
  • Glisser-déposer les favoris pour les organiser.
